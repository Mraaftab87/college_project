# Changelog

## Version 1.0.0 - January 2025

Initial release with complete inventory management features.

### What's Included

**Product Management**
- Add, edit, delete products
- Category and subcategory organization
- Product images with thumbnails
- SKU and barcode support
- Stock level tracking
- Low stock alerts

**Customer Management**
- Customer database
- Customer types (Regular, Wholesale, VIP, Corporate)
- Transaction history per customer
- Credit limit tracking

**Transactions**
- Buy/sell transaction recording
- Transaction history
- Monthly reports
- Profit/loss calculations

**User System**
- Three user roles (Admin, Manager, User)
- Role-based permissions
- Secure login
- Password recovery
- Login tracking

**Other Features**
- Barcode scanning
- Bulk CSV import/export
- Advanced search
- Responsive design
- Security features

### Security
- Password hashing
- CSRF protection
- SQL injection prevention
- XSS protection
- Secure sessions
